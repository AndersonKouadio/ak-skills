# Animation Web

Guides pour les animations **web uniquement** (Next.js / React).

**Pour le mobile (Expo / React Native)** → utiliser le skill `expo` :
- `react-native-reanimated` pour les animations (entering, exiting, layout, gestures)
- `react-native-gesture-handler` pour les interactions tactiles
- Voir [expo/building-native-ui](../expo/building-native-ui/SKILL.md) → `references/animations.md`
